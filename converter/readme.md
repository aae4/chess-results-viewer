### Подготовка к запуску

1.  **Установите библиотеки:** Убедитесь, что у вас установлены все зависимости.
    ```bash
    pip install pandas openpyxl aiohttp beautifulsoup4 lxml tqdm
    ```
2.  **Подготовьте файлы:**
    *   Положите ваш Excel-файл с турнирами в ./converter под именем `tournaments_base.xlsx`.
    *   Положите ваш большой PGN-файл в ./converter под именем `full_pgn_base.pgn`.
3.  **Запустите скрипт:**
    ```bash
    python converter.py
    ```
4.  **`converter.py`** — скрипт для **первичного создания** базы данных. Он берет большую историческую базу партий в формате PGN и базу турниров в Excel, скрапит дополнительные данные с `chess-results.com` и собирает из всего этого единый файл `database.sqlite`. Этот скрипт предназначен для запуска один раз или при полном обновлении исторической базы.

5.  **`updater.py`** — скрипт для **инкрементального обновления** данных по одному, обычно текущему, турниру. Он получает `ID` турнира с `chess-results.com`, скрапит актуальные данные (жеребьевку, результаты) и обогащает их записями ходов из локального файла `games.txt`. Этот скрипт предназначен для частого запуска во время проведения турнира. Пример запуска:
    ```bash
    python updater.py 1287717
    ```
